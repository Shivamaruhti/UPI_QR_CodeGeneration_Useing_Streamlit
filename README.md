# PayQR - UPI QR Code Generator

A Streamlit web application that generates QR codes for UPI payments.

## Features
- Generate QR codes for UPI payments
- Support for multiple UPI handles (GPay, PhonePe, Paytm, etc.)
- Optional amount field
- Download QR codes as PNG files

## Installation
```bash
# Create virtual environment
python -m venv .venv
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Usage
```bash
streamlit run app.py
```

## Supported UPI Handles
- upi
- okicici
- okaxis
- okhdfcbank
- okbizaxis
- oksbi
- paytm
- apl
